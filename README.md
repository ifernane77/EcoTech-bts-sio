# EcoTech Suivi Chantier

## Présentation du projet

EcoTech Suivi Chantier est une application web simple réalisée dans le cadre du BTS SIO option SLAM.

Le projet répond au besoin d’une entreprise de génie civil et de BTP située à Saint-Pierre, à La Réunion. L’objectif est de centraliser les informations liées aux chantiers : clients, documents, photos, interventions et rapports.

Ce projet servira de deuxième réalisation professionnelle pour l’épreuve E6 SLAM.

## Contexte

EcoTech est une petite entreprise du secteur BTP / génie civil. Elle travaille avec des particuliers, des entreprises et des promoteurs.

Actuellement, les informations de chantier sont dispersées entre plusieurs outils :

- Google Drive ;
- mails ;
- WhatsApp ;
- ordinateurs personnels.

Cette organisation peut entraîner une perte de temps lorsqu’il faut retrouver une photo, un devis, une facture, un document PDF ou un historique d’intervention.

## Objectif de l’application

L’objectif est de créer une application web interne permettant de centraliser le suivi des chantiers.

L’application doit rester simple, réaliste et adaptée au niveau BTS SIO SLAM.

## Fonctionnalités prévues

- Connexion des utilisateurs
- Gestion des rôles : administrateur, gérant, employé
- Gestion des clients
- Gestion des chantiers
- Ajout de documents PDF
- Stockage des devis et factures comme documents joints
- Ajout de photos de chantier
- Historique des interventions
- Génération d’un rapport PDF de chantier
- Droits d’accès selon les rôles

## Technologies prévues

- PHP
- MySQL
- Bootstrap
- HTML / CSS
- GitHub
- Markdown pour la documentation

## Épreuve concernée

Ce projet est principalement lié à l’épreuve E6 SLAM : concevoir et développer une solution applicative.

Il pourra aussi alimenter certaines preuves de l’épreuve E5 grâce au contexte professionnel et à l’organisation du projet.
