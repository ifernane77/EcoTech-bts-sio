# Préparation du développement — EcoTech Suivi Chantier

## Objectif

Ce document explique ce qui est préparé avant le développement réel de l’application EcoTech Suivi Chantier.

L’objectif est de rester cohérent avec la progression du BTS SIO.

---

## Situation actuelle

À ce stade du planning, les leçons de développement web ne sont pas encore étudiées.

Les notions suivantes seront étudiées plus tard :

- HTML ;
- CSS ;
- Bootstrap ;
- SQL ;
- MySQL ;
- PHP ;
- connexion à une base de données ;
- formulaires ;
- traitements serveur.

---

## Ce qui est préparé cette semaine

Cette semaine prépare :

- l’arborescence du futur projet ;
- les dossiers du futur code ;
- les pages prévues ;
- les dossiers documents et photos ;
- l’organisation générale du dépôt GitHub ;
- les preuves BTS.

---

## Ce qui sera fait plus tard

Le développement réel commencera progressivement avec :

1. les premières pages HTML ;
2. la mise en forme CSS ;
3. l’utilisation de Bootstrap ;
4. la création de la base SQL ;
5. les premières pages PHP ;
6. la connexion à la base de données ;
7. les formulaires ;
8. les fonctions de gestion des clients ;
9. les fonctions de gestion des chantiers ;
10. les tests.

---

## Pourquoi ne pas coder maintenant ?

Il est préférable de ne pas copier du code sans comprendre les notions.

Le projet doit rester cohérent avec les leçons suivies.

Avant le développement, il est donc plus sérieux de préparer :

- la structure ;
- les maquettes ;
- le modèle de données ;
- les documents techniques ;
- l’organisation du projet.

---

## Lien avec les semaines précédentes

Les semaines précédentes ont permis de préparer :

- le contexte entreprise ;
- l’expression du besoin ;
- les acteurs ;
- les flux ;
- les cas d’utilisation ;
- le modèle de données ;
- les maquettes.

La semaine 7 prépare maintenant l’organisation technique du futur code.

---

## Conclusion

Cette semaine ne correspond pas encore à du développement.

Elle correspond à une préparation technique du projet, afin de pouvoir développer proprement lorsque les leçons nécessaires auront été étudiées.
