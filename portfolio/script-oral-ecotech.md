# Script oral — EcoTech Suivi Chantier

## Objectif

Ce document prépare une courte présentation orale du projet EcoTech Suivi Chantier.

Il me servira plus tard pour l’oral et le portfolio.

---

## Présentation courte

EcoTech Suivi Chantier est une application web prévue pour une petite entreprise du BTP située à Saint-Pierre, à La Réunion.

L’objectif est de centraliser les informations liées aux chantiers.

Aujourd’hui, les informations peuvent être dispersées entre Google Drive, les mails, WhatsApp et les PC personnels.

---

## Besoin identifié

Le besoin principal est de permettre à l’équipe de retrouver plus facilement :

- les clients ;
- les chantiers ;
- les documents ;
- les photos ;
- les interventions ;
- l’avancement global des chantiers.

---

## Solution prévue

La solution prévue est une application web simple avec plusieurs rôles :

- administrateur ;
- gérant ;
- employé.

Elle permettra de gérer les clients, les chantiers, les documents, les photos et l’historique des interventions.
