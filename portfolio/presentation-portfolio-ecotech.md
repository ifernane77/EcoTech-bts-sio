# Portfolio — EcoTech Suivi Chantier

## Présentation du projet

EcoTech Suivi Chantier est un projet d’application web réalisé dans le cadre du BTS SIO option SLAM.

Le projet est lié à l’entreprise EcoTech, située à Saint-Pierre, à La Réunion.

EcoTech travaille dans le BTP, le génie civil, la rénovation et le suivi de chantier.

---

## Contexte de l’entreprise

EcoTech est une petite entreprise composée de :

- 2 gérants associés ;
- 2 employés ;
- des prestataires terrain selon les chantiers.

L’entreprise utilise plusieurs outils au quotidien :

- Google Drive ;
- mails ;
- WhatsApp ;
- PC personnels ;
- téléphones professionnels.

---

## Problème identifié

Le problème principal est que les informations ne sont pas vraiment centralisées.

Les documents, photos, devis, factures et informations de chantier peuvent se retrouver à plusieurs endroits.

Cela peut faire perdre du temps à l’équipe lorsqu’elle doit retrouver une information.

---

## Objectif de l’application

L’objectif du projet est de préparer une application simple pour centraliser le suivi des chantiers.

L’application devra permettre de gérer :

- les utilisateurs ;
- les rôles ;
- les clients ;
- les chantiers ;
- les documents PDF ;
- les photos de chantier ;
- les interventions ;
- un rapport PDF simple.

---

## Périmètre du projet

Pour rester réaliste, la première version ne prévoit pas tout.

### Fonctions prévues

- gestion des clients ;
- gestion des chantiers ;
- ajout de documents ;
- ajout de photos ;
- historique des interventions ;
- rôles admin, gérant et employé.

### Fonctions non prévues dans la première version

- accès client ;
- paiement en ligne ;
- application mobile ;
- messagerie interne ;
- synchronisation automatique avec Google Drive.
