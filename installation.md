# Installation — EcoTech Suivi Chantier

## Objectif

Ce document explique l’environnement prévu pour développer et tester le projet EcoTech Suivi Chantier.

À cette étape, l’application n’est pas encore développée. Le but est de préparer l’environnement technique avant de commencer le code.

## Environnement prévu

Le projet sera développé en local avec une stack simple :

- PHP ;
- MySQL ;
- Bootstrap ;
- Git ;
- GitHub ;
- Visual Studio Code ;
- Laragon ou un environnement local équivalent.

## Étapes prévues plus tard

Les étapes techniques prévues seront :

1. installer ou vérifier Laragon ;
2. placer le projet dans le dossier local de développement ;
3. créer une base de données MySQL ;
4. développer les premières pages PHP ;
5. tester l’application en local ;
6. versionner le projet avec Git ;
7. envoyer les modifications sur GitHub.

## Organisation prévue

Le projet prévoit :

- un dossier pour la documentation ;
- un dossier pour la base de données ;
- un dossier pour les documents PDF ;
- un dossier pour les photos de chantier ;
- un journal de bord ;
- un dépôt GitHub.
