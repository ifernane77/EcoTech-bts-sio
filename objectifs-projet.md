# Objectifs du projet — EcoTech Suivi Chantier

## Objectif général

Créer une application web simple permettant à EcoTech de centraliser les informations de suivi de chantier.

## Objectifs pédagogiques BTS SIO

Ce projet doit me permettre de travailler les compétences suivantes :

- analyser un besoin ;
- concevoir une solution applicative ;
- créer une base de données ;
- développer des interfaces web ;
- gérer des droits utilisateurs ;
- documenter un projet ;
- versionner le code avec Git et GitHub ;
- présenter une réalisation professionnelle à l’oral.

## Objectifs techniques

Les objectifs techniques prévus sont :

- mettre en place une structure PHP simple ;
- créer une base de données MySQL ;
- utiliser Bootstrap pour l’interface ;
- mettre en place une authentification ;
- gérer les rôles utilisateurs ;
- permettre l’ajout de fichiers ;
- générer un rapport PDF ;
- documenter l’installation et l’utilisation.

## Limites du projet

Pour rester réaliste, la première version ne prévoit pas :

- d’accès client ;
- de paiement en ligne ;
- de messagerie interne complexe ;
- d’application mobile ;
- de notification automatique.

Le projet doit rester simple mais suffisamment complet pour être présenté dans le cadre du BTS SIO SLAM.
