# Environnement technique — EcoTech Suivi Chantier

## Objectif

Ce document présente les choix techniques prévus pour le projet EcoTech Suivi Chantier.

## Technologies prévues

| Technologie | Rôle prévu |
|---|---|
| PHP | Développement de l’application web |
| MySQL | Stockage des données |
| Bootstrap | Mise en forme de l’interface |
| HTML / CSS | Structure et présentation des pages |
| JavaScript | Interactions simples si nécessaire |
| Git | Suivi des versions du projet |
| GitHub | Sauvegarde du code et documentation |
| Laragon | Environnement local PHP/MySQL |

## Pourquoi ces choix ?

PHP et MySQL sont adaptés à un projet BTS SIO SLAM car ils permettent de développer une application web simple avec une base de données relationnelle.

Bootstrap permettra de créer une interface propre sans passer trop de temps sur le design.

GitHub permettra de garder une trace de l’évolution du projet avec des commits.

## Organisation technique prévue

Le projet sera organisé progressivement avec :

- des fichiers de documentation Markdown ;
- une future base de données MySQL ;
- des pages PHP ;
- des dossiers pour les documents PDF ;
- des dossiers pour les photos ;
- des captures d’écran comme preuves BTS.

## Limite de cette semaine

Cette semaine ne contient pas encore de développement fonctionnel.

Elle sert uniquement à préparer l’environnement et les prérequis.
