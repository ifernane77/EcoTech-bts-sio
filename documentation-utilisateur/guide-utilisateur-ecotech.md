# Guide utilisateur — EcoTech Suivi Chantier

## Objectif

Ce document prépare le futur guide utilisateur de l’application EcoTech Suivi Chantier.

À cette étape, l’application n’est pas encore développée complètement.

Le but est de préparer les procédures principales pour les futurs utilisateurs.

---

## Utilisateurs concernés

Les utilisateurs prévus sont :

- administrateur ;
- gérant ;
- employé.

Il n’y a pas d’accès client prévu dans la première version.

---

## 1. Créer un chantier

### Objectif

Créer une fiche chantier pour centraliser les informations importantes.

### Utilisateur concerné

- administrateur ;
- gérant.

### Étapes prévues

1. Se connecter à l’application.
2. Aller dans la rubrique `Chantiers`.
3. Cliquer sur `Ajouter un chantier`.
4. Choisir le client concerné.
5. Saisir le nom du chantier.
6. Saisir l’adresse du chantier.
7. Ajouter une description simple.
8. Choisir le statut du chantier.
9. Enregistrer.

### Informations prévues

- nom du chantier ;
- client ;
- adresse ;
- statut ;
- date de début ;
- date de fin prévue ;
- description.

---

## 2. Mettre à jour le statut d’un chantier

### Objectif

Suivre l’avancement d’un chantier.

### Utilisateur concerné

- administrateur ;
- gérant ;
- employé selon les droits prévus.

### Statuts prévus

- prévu ;
- en cours ;
- en attente ;
- terminé ;
- annulé.

### Étapes prévues

1. Ouvrir la liste des chantiers.
2. Sélectionner le chantier concerné.
3. Cliquer sur `Modifier`.
4. Changer le statut.
5. Enregistrer la modification.

### Remarque

Le statut doit rester simple pour que l’équipe comprenne rapidement l’état du chantier.

---

## 3. Déposer un document

### Objectif

Ajouter un document PDF lié à un chantier.

### Utilisateur concerné

- administrateur ;
- gérant ;
- employé selon les droits prévus.

### Types de documents prévus

- devis ;
- facture ;
- plan ;
- autre document PDF.

### Étapes prévues

1. Ouvrir la fiche du chantier.
2. Aller dans la partie `Documents`.
3. Cliquer sur `Ajouter un document`.
4. Choisir le fichier PDF.
5. Indiquer le type du document.
6. Enregistrer.

### Règle prévue

Le document doit être lié au bon chantier pour être retrouvé facilement.

---
