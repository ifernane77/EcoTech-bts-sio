# Contexte entreprise — EcoTech

## Présentation générale

EcoTech est une entreprise située à Saint-Pierre, 97410, à La Réunion. Elle existe depuis environ cinq ans et intervient dans le domaine du génie civil, du BTP, de la rénovation et de la création.

L’entreprise travaille avec plusieurs types de clients :

- particuliers ;
- entreprises ;
- promoteurs.

Elle réalise notamment :

- du suivi de chantier ;
- des devis ;
- des plans ;
- des métrés ;
- du terrassement ;
- des fondations ;
- de la maçonnerie.

## Organisation de l’entreprise

L’entreprise compte :

- 2 gérants associés ;
- 2 employés ;
- des prestataires terrain selon les besoins.

Les deux gérants se répartissent le travail selon leurs disponibilités.

Un employé gère principalement l’administratif, les documents et parfois les devis.

Un autre employé suit davantage les chantiers sur le terrain.

Le contact client peut être assuré par un employé ou par un gérant. Les photos de chantier sont généralement prises par les employés.

## Situation actuelle

Aujourd’hui, les informations ne sont pas vraiment centralisées.

Les documents et informations peuvent se trouver dans :

- Google Drive ;
- des ordinateurs personnels ;
- des mails ;
- WhatsApp.

Les documents ne sont pas forcément perdus, mais l’équipe peut perdre du temps à les retrouver.

## Problème principal

Le principal problème est le manque de centralisation des informations de chantier.

Cela concerne notamment :

- les documents PDF ;
- les devis ;
- les factures ;
- les photos ;
- les échanges ;
- l’historique des interventions ;
- l’état d’avancement des chantiers.

## Besoin identifié

EcoTech a besoin d’une application simple permettant de centraliser les informations liées aux chantiers.

Cette application doit être utilisée par l’équipe interne : administrateur, gérants et employés.

Aucun accès client n’est prévu dans la première version.
