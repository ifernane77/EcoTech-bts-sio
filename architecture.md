# Architecture — EcoTech Suivi Chantier

## Objectif

Ce document présente l’organisation technique prévue pour l’application EcoTech Suivi Chantier.

L’objectif est de conserver une architecture simple, compréhensible et adaptée au niveau BTS SIO SLAM.

## Technologies utilisées

- PHP ;
- MySQL ;
- HTML ;
- CSS ;
- Bootstrap ;
- JavaScript simple ;
- Git et GitHub ;
- XAMPP ou Laragon pour l’environnement local.
