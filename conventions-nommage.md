# Conventions de nommage — EcoTech Suivi Chantier

## Objectif

Ce document définit des règles simples pour nommer les fichiers, dossiers, documents et futurs éléments de base de données.

## Fichiers Markdown

Les fichiers Markdown sont écrits en minuscules avec des tirets.

Exemples :

- `journal-de-bord.md`
- `expression-besoin.md`
- `conventions-nommage.md`

## Dossiers

Les dossiers techniques sont nommés simplement.

Exemples :

- `database`
- `storage`
- `documents`
- `photos`
- `preuves`

## Documents PDF

Les documents PDF de chantier pourront suivre ce modèle :

```text
chantier-nom-type-date.pdf
