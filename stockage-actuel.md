# Stockage actuel des informations — EcoTech

## Objectif

Ce document présente les supports actuellement utilisés par EcoTech pour stocker et échanger les informations liées aux chantiers.

## Supports utilisés actuellement

| Support | Utilisation | Limite constatée |
|---|---|---|
| Google Drive | Stockage de certains documents | Organisation variable selon les dossiers |
| PC personnels | Stockage local de fichiers | Risque de dispersion des documents |
| Mails | Envoi et réception de documents | Recherche parfois longue |
| WhatsApp | Échanges rapides et photos | Difficulté à retrouver une ancienne information |
| Téléphones professionnels | Photos, appels et messages | Photos parfois non classées par chantier |

## Problème principal

Les informations sont réparties entre plusieurs outils. Cela ne signifie pas que les documents sont perdus, mais l’équipe peut perdre du temps à les retrouver.

Le problème concerne principalement :

- les photos de chantier ;
- les devis ;
- les factures ;
- les documents PDF ;
- les échanges liés à l’avancement ;
- l’historique des interventions.

## Solution prévue

L’application EcoTech Suivi Chantier doit permettre de centraliser les informations importantes.

Chaque chantier pourra regrouper :

- les informations du client ;
- les documents PDF ;
- les devis et factures stockés comme fichiers joints ;
- les photos ;
- l’historique des interventions ;
- le statut du chantier ;
- un rapport PDF généré.

## Bénéfices attendus

La centralisation permettra :

- un gain de temps ;
- une meilleure organisation ;
- une meilleure visibilité sur l’avancement ;
- une réduction des recherches dans plusieurs outils ;
- une meilleure traçabilité des interventions.
